The Problem

Markdown editing in VS Code is broken

You open a markdown file and immediately split your screen -- source on the left, preview on the right. Every edit means mentally translating between raw syntax and rendered output. Your eyes bounce between panes. Your focus fragments.

Managing multiple markdown files makes it worse. Links break silently. There is no easy way to find what references what. Simple tasks like inserting a table or pasting an image become a context-switching exercise that pulls you out of your writing flow.

Features

Everything you need to write Markdown

Edit rendered Markdown directly. No split panes, no mental translation.

WYSIWYG Editing

Write in a fully rendered view. Bold, italic, headings, lists, and more -- all visible as you type, exactly as they will appear.

Smart Paste

Paste URLs onto selected text to create links. Paste images directly into the document. Paste HTML and get clean Markdown.

Link Intelligence

Autocomplete links to workspace files. Detect broken references automatically. View a backlinks panel showing what links to the current file.

Table Editing

Insert, resize, and edit tables visually. Add or remove rows and columns without wrestling with pipe characters and alignment syntax.

Source Mode Toggle

Switch instantly between the WYSIWYG view and raw Markdown source. Inspect and fine-tune syntax when you need precise control.

Theme Toggle

Switch between light and dark themes within the editor. The preview always matches your preferred reading environment.

Find & Replace

Search through your document and replace text without leaving the editor. Full in-editor find and replace, right where you are writing.

Code Highlighting

Fenced code blocks render with syntax highlighting for dozens of languages. Inline code is styled distinctly for clarity.

Export & Print

Export your document as HTML or print it directly from the editor. Share your work in formats beyond Markdown source.

Condensed Toolbar

A compact, contextual formatting toolbar that stays out of your way. All the controls you need without the clutter.

What Sets MikeDown Apart

Features no other VS Code Markdown editor has

These capabilities are unique to MikeDown.

Broken Link Detection

Automatically scans your document for links that point to missing files or invalid targets. No more publishing pages with dead links.

Link Autocomplete

Start typing a link and MikeDown suggests matching files from your workspace. Navigate between documents without memorizing paths.

Backlink Panel

See every file that links to the document you are editing. Understand the structure and connections within your Markdown content.

In-Editor Find & Replace

A full find-and-replace experience built into the WYSIWYG view. Other extensions force you to switch to source mode first.
